# ASP.NET Core BFF Example

This project is a minimal Backend-for-Frontend (BFF) using ASP.NET Core Web API. It is designed for code review, static code analysis, and architecture assessment.

## Features
- ASP.NET Core Web API
- Example controller, service, and model
- Follows best practices for maintainability and extensibility

## Prerequisites
- [.NET 8 SDK](https://dotnet.microsoft.com/en-us/download)

## Getting Started

1. **Restore dependencies:**
   ```sh
   dotnet restore
   ```
2. **Build the project:**
   ```sh
   dotnet build
   ```
3. **Run the project:**
   ```sh
   dotnet run
   ```

The API will be available at `http://localhost:5000` (or as indicated in the output).

## Project Structure
- `Controllers/` - API controllers
- `Services/` - Business logic/services
- `Models/` - Data models

## Code Quality
This project is intended for static code analysis and architecture review. Feel free to extend it for your assessment needs.
